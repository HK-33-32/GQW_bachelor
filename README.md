# 6-DOF Robotic Manipulator with Adaptive Control

Дипломный проект (ВКР), посвящённый разработке 6-осевого
роботизированного манипулятора с адаптивной системой управления,
способного работать в динамически изменяющейся среде и предотвращать
столкновения.

## О проекте

В рамках работы разработан робот-манипулятор с интеллектуальной системой
управления, ориентированный на: - работу в неструктурированной среде -
обход препятствий в реальном времени - предотвращение столкновений -
взаимодействие с человеком и роботизированными системами

Основой проекта является:
- Кинематическое моделирование
- Разработка аппаратной части
- Алгоритмы управления
- Симуляция в ROS2 + Gazebo

## Цели и задачи

Цель: Разработка 6-осевого манипулятора с адаптивной системой
управления.

Основные задачи:
- Создание кинематической модели манипулятора
- Разработка алгоритмов управления и планирования траектории
- Реализация системы обнаружения препятствий (LiDAR)
- Разработка электрической схемы
- Создание симуляционной модели
- Тестирование алгоритмов

## Основные характеристики

-   Количество осей: 6
-   Грузоподъёмность: до 8 кг
-   Точность позиционирования: ≤ 1 мм
-   Скорость перемещения: ≥ 500 мм/с
-   Рабочая температура: 0...+40°C

## Ключевые особенности

Адаптивное управление:
- Реакция на изменения среды в реальном времени
- Перестроение траектории при обнаружении препятствий

Обнаружение столкновений:
- Использование LiDAR
- Алгоритмы Collision Detector и Dead Area Detector

Планирование траектории:
- Метод потенциальных полей
- Учет препятствий и ограничений суставов

Симуляция:
- Gazebo Sim
- ROS2 (Jazzy)
- RViz

## Кинематика

Использована модель Денавита--Хартенберга: - 6 степеней свободы -
Ограничения по углам - Полная кинематическая цепь

## Симуляция

Реализовано:
- Импорт модели в Gazebo
- Настройка SDF/URDF
- Подключение плагинов

## Алгоритмы системы

Основные модули:
- Robot_Model_Publisher
- TF_Listener
- Joint_State_Aggregator
- Lidar_Filter
- Lidar_Processor
- Collision_Detector
- Dead_Area_Detector

## Аппаратная часть

Основные компоненты:
- Шаговые двигатели
- Гармонические редукторы
- LiDAR Unitree L1
- Драйверы HBS860H
- Контроллер Jetson Orin NX

## Энергопотребление

Общая мощность системы: ≈ 2465 Вт

## Сборка и запуск

    source /opt/ros/jazzy/setup.bash
    mkdir -p ~/ros2_ws/src
    cd ~/ros2_ws/src
    ros2 pkg create --build-type ament_cmake robot_project
    cd ~/ros2_ws
    colcon build
    source install/setup.bash
    ros2 launch <package_name> <launch_file>
